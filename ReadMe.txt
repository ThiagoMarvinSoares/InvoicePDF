# Automotive Window Film Invoice Maker

This repository contains a simple invoice maker specifically designed for companies dealing in automotive window films. The invoice maker is built using PDFKit, a PDF generation library for Node.js.

## Features

- Generate professional-looking invoices for automotive window film services.
- Customizable invoice template to include company logo, details, and service descriptions.
- Easy to use and integrate into existing systems.

## Installation

Ensure you have Node.js installed on your system. Then, clone this repository and install the dependencies:
